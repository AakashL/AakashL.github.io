Cascading Style Sheets, or CSS, is a markup language used to edit the look for formatting of an HTML page. Using CSS versus just HTML allows you to make a variety of changes and tweaks to your site by editing the CSS stylesheet,
In CSS, each HTML element, such as p, h1, etc. , can be assigned either an ID or a Class. Think of this as a nametag for that element. So what's the difference?
IDs are meant to be unique, meaning that each element can have only one ID, and each page can only have one element with a specific ID.
On the other hand, the same class can be used on multiple elements, and elements can even have multiple classes. This is best for any sort of styling that needs to be applied to multiple objects, even on multiple page(s).
