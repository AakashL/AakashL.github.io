SQL Injection

September 9, 2015

After having learned about the awesomeness of SQL databases, it's easy to see how they are an integral part of tons of sites which store tables of information. Unfortunately, since this technology is so widely used, it's also a big target for hackers. SQL injection, a technique hackers use to exploit SQL databases, and behind 97% of breaches!

Using SQL injection: hackers can read and modify data, execute administrative commands, and recover content. Many sites allow legitimate users to submit and retrieve information using tools such as login forms, feedback forms, search pages, shopping carts, etc. however these same tools can potentially be exploited with SQL injection. If the proper checks aren't in place, malicious users can use these areas of a site to execute their own SQL code on your database, leading to very undesirable results. Make sure to check out some of the recourses below for more information and examples of SQL injection.
