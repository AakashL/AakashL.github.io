Last week I wroite a bit about arrays and hashes in Ruby. This week I'll be writing about enumerables which allow you to access, edit, add, and delete elements within hashes and arrays. The .map method iterates through an object, and applies whatever commands you have in your block to each element.

my_array =[1,2,3]
double_array = my_array.map {|i| i*2} #> [2, 4, 6]

In the above example, we call the .map method on the array my_array. Each object passted is temporarily assigned to the variable i, and the instructions in the block, which in this case are i*2, are applied.
