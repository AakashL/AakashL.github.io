Javascript Prototypes & Constructors

Switching from Ruby to JS and then back again was a little jarring at first, especially since JS has fewer built in methods. Finding parallels between the two languages has really helped me get a better understand of and more easily switch between the two. On this page will be a cheat-sheet on JS constructors, and links to resources I found to be helpful.

Creating a Constructor Function


			function user(first, last, join_date, referral_site) {
				this.firstName = first;
				this.lastName = last;
				this.joinDate = join_date;
				this.referral_site = referral_site
			}

Creating new objects


				var user1 = new user("Aakash", "Lakhotia", "01-08-2012", "TRiBot");
				var user1 = new user("John", "Conrad", "08-22-2014", "OSBot");

Adding methods

To a prototype


			function user(first, last, join_date, referral_site) {
				this.firstName = first;
				this.lastName = last;
				this.joinDate = join_date;
				this.referral_site = referral_site
				this.sayName = function() {
					return 'Hi, my name is ' + this.firstName + ' ' + this.lastName
				}
			}


			var user1 = new user("Aakash", "Lakhotia, "01-08-2012", "TRiBot");

			user1.sayName
			//'Hi, my name is Aakash Lakhotia'
			To an object

			user1.name = function () {
				return this.firstName + " " + this.lastName;
			}

Adding properties

To a prototype


			function user(first, last, join_date, referral_site) {
				this.firstName = first;
				this.lastName = last;
				this.joinDate = join_date;
				this.referral_site = referral_site
				this.ref_game = "Runescape"
				}
			}

To an existing object


				user1.ref_game = "Runescape"
